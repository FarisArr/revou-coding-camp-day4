# Productivity Dashboard

Productivity Dashboard adalah website sederhana yang dibuat untuk membantu pengguna meningkatkan produktivitas dalam mengatur waktu belajar atau bekerja. Website ini menyediakan fitur timer fokus, manajemen tugas, serta quick links untuk mengakses website penting dengan cepat.

Project ini dibuat sebagai bagian dari latihan dalam mengikuti program Software Engineering dari RevoU.

## Fitur Utama

- Menampilkan waktu dan tanggal secara real-time
- Focus Timer (Pomodoro 25 menit)
- Menambahkan dan mengelola daftar tugas (Task List)
- Mengedit dan menghapus tugas
- Menambahkan quick links ke website penting
- Tampilan dashboard yang sederhana dan user-friendly

## Teknologi yang Digunakan

- HTML
- CSS
- JavaScript
